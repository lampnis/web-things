1. Dynamic Array (ArrayList)

▼

A dynamic array is an array that can grow or shrink in size. It provides O(1) random access.

# Python list is dynamic by default
my_list = [1, 2, 3]
my_list.append(4)  # O(1) amortized
print(my_list)
// Simple dynamic array implementation in C
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int *array;
    size_t size;
    size_t capacity;
} DynamicArray;

void initArray(DynamicArray *a, size_t initialCapacity) {
    a->array = malloc(initialCapacity * sizeof(int));
    a->size = 0;
    a->capacity = initialCapacity;
}

2. Singly Linked List

▼

A linear data structure where elements are stored in nodes, each pointing to the next.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None
struct Node {
    int data;
    struct Node* next;
};

void printList(struct Node* n) {
    while (n != NULL) {
        printf("%d ", n->data);
        n = n->next;
    }
}
