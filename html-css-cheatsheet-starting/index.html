Basic HTML

First line always is <!DOCTYPE html>. Basically, everything is based on opening <tag> and closing </tag>. In the <head> of the HTML document you add all the links to other files like stylesheets, fonts, scripts etc. For example, see the code snippet of this page below:

                        
<!DOCTYPE html>
<html>
<head>
    <title>Cheats</title>
    <link rel="stylesheet" href="./styles.css" type="text/css" />
    <link rel="stylesheet" href="../styles/prism.css" type="text/css" />
    <script src="../styles/prism.js" type="text/javascript"></script>
</head>
<body>
    <header>
        <h1>HTML & CSS Cheatsheet</h1>
        <nav>
            <ul>
            .
            .
            .
            </ul>
        </nav>
    </header>
<body>

Syntax

HTML syntax based on opening and closing tags. In head define used stylesheets, scripts etc. CSS syntax is little bit different. Here below, I show a useful example in designing and positiong the elements of the page (that code adds a red border for each element on the page). We will continue on details about classes, ids and different properties of objects that we can customize using CSS. 

                        
* {
    border: 1px solid red;
}
